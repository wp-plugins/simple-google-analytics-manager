Enables Google Analytics on all pages.

== Description ==

This plugin adds the required javascript for google analytics.

For more information visit:

[Google Analytics Manager](http://www.google.com/analytics)

== Installation ==

1. Upload `gamanager` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the UA from Google Analytics (UA-XXXXXXX) to the settings (Admin > Settings > Google Analytics Manager)
